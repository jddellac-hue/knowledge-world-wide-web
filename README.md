# knowledge-world-wide-web

> ## ⚠️ Disclaimer — base de connaissances produite et maintenue par IA
>
> Ce contenu est une **base de connaissances techniques générique**, **produite,
> enrichie et maintenue par un assistant IA** (agents type Claude Code / Copilot)
> en dialogue itératif avec son auteur, à partir de **sources publiques**
> (documentation officielle, standards, ouvrages SRE/DevOps, articles techniques).
>
> **Doctrines de maintenance appliquées :**
> - **LIRE → COMPRENDRE → RÉÉCRIRE** : jamais de copie brute ; reformulation
>   structurée, sourcée, autosuffisante.
> - **Souveraineté & anonymisation** : aucune donnée d'organisation, aucun secret,
>   aucun nom réel d'entreprise / personne / infrastructure interne — uniquement
>   du savoir portable. Tout élément spécifique à une organisation est maintenu
>   séparément, hors de ce dépôt.
> - **Sourcing** : chaque pattern ou chiffre non trivial est référencé.
> - **Vision « at now »** : la doc décrit l'état courant des bonnes pratiques ;
>   les contenus obsolètes sont retirés (l'historique git garde la trace).
>
> À **challenger** : ce n'est pas une vérité établie mais une synthèse vivante,
> faillible, à confronter à votre contexte et aux sources primaires.

Connaissances techniques **génériques et publiques** — obtenues ou obtenables depuis internet.
Compatible avec tout agent IA : Claude Code, GitHub Copilot, Cursor, Aider.
